# www.lukemorton.co.uk

## Developing

**Install dependencies:**

```
npm install
```

**Compile data files:**

```
npm build
```

**Run development server:**

```
npm run dev
```

The app will be served on [http://localhost:3000](http://localhost:3000)

**Run tests and linting:**

```
npm test
npm run lint
```

## Deploying

**Deploy to now.sh:**

```
npm run deploy
```

**Cleaning up old deploys:**

```
npm run clean-deploys
```
